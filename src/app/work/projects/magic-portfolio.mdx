# Ikhtisar

Magic Portfolio adalah template portofolio yang memudahkan kamu membuat portofolio dengan desain elegan dan responsif. Template ini dibangun menggunakan Once UI untuk NextJS dan menampilkan fitur-fitur berikut:

- **Desain Responsif:** Tampilan optimal di berbagai perangkat.
- **SEO Otomatis:** Generasi metadata dan open-graph otomatis.
- **Konten Fleksibel:** Dukungan untuk halaman seperti blog, karya, galeri, dan tentang/CV.
- **Kustomisasi Mudah:** Penyesuaian melalui atribut data.
- **Lokalisasi:** Mendukung portofolio multibahasa menggunakan library next-intl.
