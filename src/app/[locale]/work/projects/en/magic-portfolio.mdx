# Overview

Magic Portfolio is a portfolio template that helps you create a portfolio with an elegant, responsive design. This template is built with Once UI for NextJS and offers the following features:

- **Responsive Design:** Optimized for various devices.
- **Automatic SEO:** Automatic metadata and open-graph generation.
- **Flexible Content:** Supports pages like blog, work, gallery, and about/CV.
- **Easy Customization:** Customization through data attributes.
- **Localization:** Supports multilingual portfolios using the next-intl library.
