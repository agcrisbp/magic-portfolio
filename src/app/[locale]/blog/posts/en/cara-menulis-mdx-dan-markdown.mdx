# Summary

Some of the MDX and MD writing formats here are templates that have been modified by **[Aghea]({baseURL}/github)**, not the original version from [Once UI]({baseURL}/once-ui). You can access the repository for this template at [github/agcrisbp/magic-portfolio]({baseURL}/magic-portfolio).

If you choose not to use MDX (Markdown JSX), you can use regular Markdown (MD) with a few exceptions.

- Update: January 16, 2025
  - [LaTeX](#latex).
  
- Update: January 18 & 19, 2025
  - [Table](#table): Added `collapse` property to automatically adjust table width.
  - [QuoteBlock](#quoteblock): Added special properties for [Information](#2-informasi), [Important](#3-penting), and [Warning](#4-peringatan).
