# Ikhtisar

Beberapa format penulisan MDX dan MD di sini adalah templat yang sudah dimodifikasi oleh **[Aghea]({baseURL}/github)**, bukan versi asli dari [Once UI]({baseURL}/once-ui), repository untuk templat ini dapat kamu akses di [github/agcrisbp/magic-portfolio](https://github.com/agcrisbp/magic-portfolio).

Jika kamu memilih untuk tidak menggunakan MDX (Markdown JSX), kamu dapat menggunakan Markdown biasa (MD), dengan beberapa pengecualian.

- Pembaruan: 16 Januari 2025
  - [LaTeX](#latex).
  
- Pembaruan: 18 & 19 Januari 2025
  - [Table](#table): Menambahkan properti `collapse` untuk mengatur lebar table secara otomatis.
  - [QuoteBlock](#quoteblock): Menambahkan properti khusus untuk [Informasi](#2-informasi), [Penting](#3-penting), dan [Peringatan](#4-peringatan).
  - Perbaikan bug, pembaruan UI, membuat fitur `Share Button` dan [InlineImages](#inline-images).
